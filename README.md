# My Amazon Sample Clone 👋

**🚀 Live Demo: <a href="https://sriharshitha-konkathi.github.io/Amazon-Clone/" target="_blank" rel="noopener noreferrer">View Project on GitHub Pages</a>**

Hi there! This is a sample Amazon clone I built to practice and showcase my frontend development skills.

## 🛠️ What's inside?
Here's a quick look at the tech stack I chose to build this project:

- **React** for building out the UI components.
- **Redux Toolkit** to manage all the state.
- **Tailwind CSS** for keeping the styling fast, clean, and responsive.
- **React Router** to handle moving around different pages smoothly.
- **Parcel** as my lightning-fast bundler.
- **Jest & React Testing Library** to make sure everything works perfectly.

## 🚀 How to run it locally
Want to try it out on your own machine? It's super easy!

1. Install the dependencies:
   ```bash
   npm install
   ```
2. Start up the dev server:
   ```bash
   npm start
   ```

*P.S. If you want to check out the tests, just run `npm test` or `npm run watch-test`.*
